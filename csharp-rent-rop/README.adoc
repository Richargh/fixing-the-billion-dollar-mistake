= Classic vs Railway-Oriented Programming (ROP) in C#

== Tech Stack

* C#9 and .NET 5
* Xunit

== Usage

. Run classic console: `dotnet run -p classic`
. Run ROP console: `dotnet run -p rop`
. Run tests: `dotnet test`
